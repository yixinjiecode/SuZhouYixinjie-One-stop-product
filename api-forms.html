3.2. Form¶

能批量对多个控件进行赋值、取值、重置、验证、获取错误信息等。 节省大量针对单独控件的操作代码。 加载表单：1）使用ajax获取数据；2）将数据一次性设置给多个控件：form.setData(obj); 保存表单：1）获取多个控件数据：var obj = form.getData()；2）使用ajax提交到服务端保存。

Usage

提交表单数据：

var form = new nui.Form("#form1");
var data = {forms:form.getData()};      //获取表单多个控件的数据
var json = nui.encode(data);   //序列化成JSON
nui.ajax({
    url: "org.gocom.components.nui.demo.TForm.saveForm.biz.ext",
    type: "post",
    data: json ,
    contentType: "text/json",
    success: function (text) {
        alert("提交成功，返回结果:" + nui.decode(text.returnForms));
    }
});

加载表单数据：

var form = new nui.Form("#form1");
nui.ajax({
    url: "org.gocom.components.nui.demo.TForm.loadForm.biz.ext",
    type: "post",
    success: function (text) {
        var data = nui.decode(text);   //反序列化成对象
        form.setData(data);             //设置多个控件数据
    }
});
